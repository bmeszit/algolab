# 3\. labor

Algoritmusok átismétlése:

- Páros gráfban maximális párosítás: javítóutas algoritmus.
- DSU + union by size trükk + path compression. Kruskal.

1\. [Tánc](./ora1-tanc/) 2. +5 pont (Lilla, Matyi, P. Ákos, Ádám)
